My Markdown Cheat Sheat

This is my personal markdown cheat sheet. Markdown and its limitations kinda suck, but it's pretty simple, relatively universal, and much better than HTML for writing with simple formatting.

Originally published to:

  • https://deliverystack.net/2025/11/30/my-markdown-cheat-sheet/
To Get ThisDo This
HeadingsStart headings with # characters to match level (##, ###, etc.)
Links[text](url)
Images![<text>](<URL> "<title>")
Preformat`<inline>` or ```<ENTER><multiline><ENTER>```
Special Characters`Preformat` or escape with a backslash (\\) (`` ` `` to escape a backtick in preformat)
Quote> <text>
Italics*<text>*
Bold**<text>**
Bold Italics***<text>***
Avoid trailing line breakAppend two spaces
Numbered list1. prefix
Bullets- prefix
Nested bulletsIndent four spaces
Horizontal rule--- (or more characters)
&nbsp;Empty table cell
UnderlineDon't even try it

Tables

| Heading | Heading  
| --------|--------   
| Value   | Value
  • Spacing and the number of dashes in the separator doesn't matter.
  • Thou shalt not consider adding linefeeds within a table cell.
