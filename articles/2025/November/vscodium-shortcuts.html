VSCodium Keyboard Shortcuts

This blog post lists some keyboard shortcuts that I would like to memorize for frequent use in VSCodium. These shortcuts likely work in Visual Studio Code and possibly other Integrated Development Environments (IDEs) as well. This post also provides links to resources with additional relevant information.

Originally published to:

  • https://deliverystack.net/2025/11/25/vscodium-keyboard-shortcuts/

VSCodium Keyboard Shortcuts

SequenceFunction
CTRL+k,vVisualize markdown
CTRL+fFind in file
CTRL+ALT+fFind in project
ALT+zToggle word wrap
CTRL+k,zToggle zen mode
CTRL+= | Ctrl+-Zoom in | Zoom out
F11Toggle fullscreen mode
CTRL+k,CTRL+sConfigure keyboard shortcuts
CTRL+k,CTRL+tToggle light/dark theme
CTRL+0Focus sidebar
CTRL+TABCycle open files
CTRL+SHFT+eExplorer view.
CTRL+SHFT+gGit view.
